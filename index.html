This is a yum repository for the yumrepos rpms and its centos7 configuration!
Its Metadata can be found at repodata/repomd.xml.

Access via 

yum -c yumrepos.conf install yumrepos-behind-nginx-on-centos7
with the following yumrepos.conf file (download yumrepos.conf directly or copy-n-paste the following code snippet): 

[main]
plugins=1

[yumrepos]
name=the home of the yum repos
baseurl=https://arnehilmann.github.io/yumrepos/
gpgcheck=0
sslverify=1
enabled=1
