YumRepos Repo

This is a yum repository for the yumrepos rpms and its centos7 configuration!
Its Metadata can be found at repodata/repomd.xml.

Prerequisites

  • centos7 machine
  • including root access
  • internet access

Installation

  1. enable epel repos (as root user): yum install epel-release
  2. enable this repo by putting yumrepos.repo into /etc/yum.repos.d/.
  3. install yumrepos configuration for centos7: yum install yumrepos-behind-nginx-on-centos7
  4. generate first key/secret api-key: generate-yumrepos-key-pair
